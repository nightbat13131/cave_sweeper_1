Game: Minesweeper-like

minesweeper rules to avoid dangerous tiles - 
Use repels to chase off danger and get it's loot. 
Can only walk where you can see. 
Advance through the caves to get as deep as posible gathering a much loot as possible. 

Zubat and repell from Pokemon. I don't own pokemon.
